Projects

  • Food Truck Website
    Developed and designed a user-friendly food truck website that allows customer ordering. The site includes an interactive home page where users can input their name and phone number for easy ordering. After inputting their information, users are directed to a detailed menu screen, where they can browse and add food items to their order. An order ticket on the right side of the menu keeps track of selected items and quantities in real-time. After finalizing their order, customers can submit their selections, which will be stored in a database. This allows users to view their order history when revisiting the site.
  • To-Do List Location App
    Developed a to-do list app that supports multiple users, designing a database to manage user data exclusively and efficiently. The app allows users to create and manage multiple to-do lists, adding tasks with descriptions, location, and radius using a map interface friendly to the user. Tasks can be assigned to the user or to friends, with the added functionality of sending location-based notifications when the user enters a specific radius. Additionally, the app includes a messaging feature, adding communication between users. The app offers an interactive way to manage tasks while allowing collaboration with friends.
